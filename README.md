# 🌐 Personal Portfolio Website

A modern, responsive, and professional personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, achievements, and contact information while providing a clean and interactive user experience.

- Responsive Design
- Modern UI/UX
- About Me Section
- Skills & Technologies
- Contact Form with EmailJS Integration

```bash

```

3. Open `index.html` in your browser.
